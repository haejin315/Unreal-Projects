# 용어 정리
액터 : 화면 상에 있는 모형, 도형 같은 것들<br>
기즈모 : 액터 클릭하면 나오는 화살표, 움직여서 액터 이동 가능, e는 회전 w는 평행이동 r는 스케일 변경, ALR + 마우스 휠 동시에 눌러서 기즈모의 위치 변경 가능, 피벗 오프셋으로 설정해야 고정됨, 특성 리소스의 기즈모 위치를 바꾸는 방법은 왼쪽위 모드를 모델링 모드, 피벗 선택 <br>
스낵 : 특정 비율대로 수정됨, 화면 오른쪽 위에서 설정 가능<br>
텍스쳐라는 이름의 이미지를 활용해서 머테리얼을 만들고 StaticMesh의 외형을 결정하는 재질을 입힐 수 있다.<br>
